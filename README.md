# meteo-tactique

Application météo en temps réel développée en Python.
Affiche les conditions météorologiques actuelles d'une ville via une API.

## Fonctionnalités

- Recherche météo par ville
- Affichage de la température, humidité et conditions
- Interface en ligne de commande

## Installation

```bash
pip install -r requirements.txt
python app_meteo.py
```
